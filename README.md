## About The Project
Tickitz backend made with express js, nodejs, mysql

### Feature
- CRUD on movies
- CRUD on schedule
- CRUD on location
- CRUD on cinema
- Login/Register
- Search movies
- Sort movie by genre

## Getting Started

Get started with this project, intructions on setting up your project locally.<br />
To get a local copy up and running follow these simple steps.
### Prerequisites

Before installing, you must be install [node js](https://nodejs.org) 
### Installation

1. Clone this repo
 
```sh
git clone https://github.com/GygaMaulana/tickitz_backend_week4.git
```
2. Install node_modules

```sh
npm install
```
3. Install express js

```sh
npm i express
```
4. Install mysql

```sh
npm i mysql
```
5. Start the project

```sh
npm start
```
